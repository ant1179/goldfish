## Requirements

This application is a note taking app that allows you to take notes, search for notes, summarize notes, and create notes using AI.
AI should remain optional and non intrusive if the user prefers to handle his notes manually.

The different features of the application are listed in this file as user stories.
User stories are end to end features which should be implemented in the application.

## User Stories

### User story 1: 

#### Description

- As a user, I want to be able to take notes and persist them in the database.

#### Acceptance Criteria

- The user should be able to take notes and persist them in the database.
- Note taking should be possible through a web interface.
